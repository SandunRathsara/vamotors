## What Was Built

### reconciliation-table.tsx
- Removed `useQueryStates` with page/pageSize/status/source params
- Added `React.useState<DataTableState>` with default `{ page: 1, perPage: 20, sortBy: "", sortDir: "desc" }`
- Retained `status` and `source` as standalone `useQueryState` page-level filters
- Added `onStateChange={setTableState}` to DataTableShell
- Removed `isFiltered`/`onClearFilters` from DataTableShell props
- Preserved Sheet/dialog form for reconciliation, SourceBadge component, formatDate helper, buildColumns factory

### report-table.tsx (generic — covers all 7 report pages)
- Removed `useQueryStates` with page/pageSize/sortBy/sortDir/dateFrom/dateTo
- Added `React.useState<DataTableState>` with default `{ page: 1, perPage: 20, sortBy: "", sortDir: "asc" }`
- Retained `dateFrom` and `dateTo` as standalone `useQueryState` page-level filters
- Added `onStateChange={setTableState}` to DataTableShell
- Removed `isFiltered`/`onClearFilters` from DataTableShell (kept them in component for the "Clear dates" button above the table)
- Preserved Breadcrumb, PageHeader, date range filter JSX, loading state

### app/(app)/cash-flow/page.tsx
- Removed `useQueryStates` with page/pageSize/sortBy/sortDir/type/dateFrom/dateTo
- Added `React.useState<DataTableState>` with default `{ page: 1, perPage: 20, sortBy: "", sortDir: "desc" }`
- Retained `type`, `dateFrom`, `dateTo` as standalone `useQueryState` page-level filters
- Added `onStateChange={setTableState}` to DataTableShell
- Removed `isFiltered`/`onClearFilters` from DataTableShell (kept for the "Clear filters" button above the table)
- Preserved all summary cards (Total Inflow, Total Outflow, Net Position), PageHeader, CurrencyDisplay, StatusBadge
