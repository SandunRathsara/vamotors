## What Was Built

**Lease Deals** (`/lease-deals`): DataTableShell with columns Date, Customer (resolved from customerFixtures), Vehicle, Finance Company (resolved from thirdPartiesFixtures), Type badge, Loan Amount, Status. "New Lease Deal" dialog opens inline form. Detail page at `/lease-deals/[id]` renders 5 Tabs:
- Overview: all deal fields in grid
- Customer & Guarantors: customer info + guarantors Table with eligibility
- Documents: customer + vehicle doc checklists (ChecklistItem), inspection photo grid (9 slots, ApplicationProcessing only)
- Cash Advance: facility status, amount, mock blockers and delay inquiries
- Activity Log: vertical timeline

**Lease Comparison** (`/lease-comparison`): 4 inputs (Vehicle Model, Year, Loan Amount, Period) trigger `/api/lease-comparison`. Results render as side-by-side Cards with installment/period/total. "Proceed" button opens in-page panel with document fees + government fees inputs and total-with-charges display.

**Rate Sheets** (`/lease-rate-sheets`): Finance company Select filter hits `/api/lease-rate-sheets`. Two Tables rendered: Vehicle Model × Year (max loan amounts) and Loan Amount × Period (monthly installments). Empty cells show "—".

**Lease Dispatch** (`/lease-dispatch`): Two Cards — Pending Dispatch table and Dispatched table. "Mark Dispatched" button opens Dialog with optional tracking number input. PATCH to `/api/lease-dispatch`. Toast "Documents marked as dispatched." Empty state: "Nothing awaiting dispatch."

**Commission Reconciliation** (`/lease-reconciliation`): Paginated DataTableShell with Status + Source Select filters. Columns: Date, Source badge, Entity, Deal Ref, Expected Commission, Received Commission, Discrepancy (red when positive), Status, Actions. "Reconcile" button opens Dialog for received amount. PATCH calculates discrepancy server-side. Toast "Commission reconciled."
