## Key Patterns

- `MockStore<T>` generic class: `getAll`, `getById`, `create`, `update`, `delete`, `query`
- `query()` supports: pagination (page/pageSize), sorting (sortBy/sortDir), full-text search (q), exact-match filters
- All monetary amounts as integer cents — `formatCurrency(350000000)` → `"Rs. 3,500,000.00"`
- `injectLatency()` reads `MOCK_LATENCY_MS_MIN` / `MOCK_LATENCY_MS_MAX` env vars (default 200–600ms)
- Settings is a singleton `SettingsStore` (not a collection)
