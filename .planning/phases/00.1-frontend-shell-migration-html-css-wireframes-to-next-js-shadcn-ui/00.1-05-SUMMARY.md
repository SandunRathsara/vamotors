## Decisions Made

1. **TanStack Form v1 type inference** — `useForm<T>` requires all 12 generic args when explicit. Fix: declare `const EMPTY_VALUES: MyFormValues = {...}` then pass `defaultValues: EMPTY_VALUES` — TypeScript infers `TFormData` from the object type.

2. **PurchaseBody type** — The purchase forms send `mileageAtPurchase` (not on `Vehicle` schema) for seeding the mileage history on creation. Route handler uses `type PurchaseBody = Partial<Vehicle> & { mileageAtPurchase?: number }`.

3. **Brand-New tab** — VIN and CR Number fields are intentionally absent per plan requirement VPRC-03 (not required for brand-new vehicles).
